# 隱私權保護政策

*2022年6月30日第003次 管理會議制定*  
*2023年10月10日第007次 修訂通過*  
*生效日期：2023年11月10日*

## 第一章 介紹及說明

CloudXact 一向高度重視您的隱私，並致力於採取最嚴謹的措施來保障您的個人資訊安全。本隱私政策詳細說明我們如何收集、使用、處理和分享您的個人資訊，並列舉我們在保護資訊安全和履行責任方面的承諾。如果您不同意本政策中的任何條款，請即時停止使用 CloudXact 的相關服務，並聯繫我們以獲取更多說明。

以下是相關術語的定義與適用範圍，旨在幫助您更清楚了解我們的操作方式：

- (a)**「客戶」、「您」或「您的」**：指任何訪問我們網站、使用服務或接受本政策條款的個人或法人，包括消費者及企業用戶；
- (b)**「公司」、「我們」或「CloudXact」**：指 CloudXact 及其子公司、附屬公司和合作夥伴；
- (c)**「一方」、「各方」或「我們」**：指公司與客戶雙方，或其中任意一方。

本隱私政策適用於 CloudXact 所提供的所有服務，包括但不限於數位平台、移動應用程式與硬體設備解決方案。我們將根據適用法律與規範為所有用戶提供清晰、透明的隱私保護標準和服務保障範疇。

## 第二章 使用我們的服務

1. 連結審核與批准條件
   CloudXact 保留批准某些組織或企業連結至其網站的權利，並基於以下條件進行審核：

   - (a) 連結行為不得損害 CloudXact 或其關聯企業的品牌形象，並需符合公共利益；
   - (b) 申請連結的組織或企業應在 CloudXact 的記錄中無不良信用紀錄，並具備良好的商譽；
   - (c) 該連結需與 CloudXact 的核心業務需求及利益一致；
   - (d) 使用連結的方式應合乎合理範疇，並避免誤導性陳述或不正當行為。
2. 連結的使用準則
   符合條件的組織或企業需遵守以下準則方可連結至 CloudXact 網站：

   - (a) 所提供的信息需透明、清晰，不得對用戶造成誤導；
   - (b) 不得虛假暗示 CloudXact 支持或背書該組織的產品、服務或行為；
   - (c) 連結的網站內容需與 CloudXact 相關，尤其在技術、教育或其他有明顯編輯價值的領域應具備高度相關性。
3. 連結申請程序
   有意連結至 CloudXact 網站的組織或企業，請通過以下方式提交申請，並提供完整的相關資訊：

   - (a) 聯絡人的姓名及組織名稱；
   - (b) 可聯繫方式（如電子郵件地址及電話號碼）與網站 URL；
   - (c) 預計連結的頁面及其具體描述；
   - (d) 連結目的及其與 CloudXact 業務或服務的相關性。

   請將上述資訊發送至 [privacy@cloudxact.com](mailto:privacy@cloudxact.com) ，我們將在收到申請後進行審核，並在最慢兩週內與您聯繫確認審核結果。

## 第三章 隱私與版權保護

1. **版權保護承諾**
我們嚴格遵守相關法律與規範，致力於保護所有用戶的版權及知識產權。對於涉嫌版權侵權的情況，我們將依法律程序處理，並對屢次違規者採取適當措施，包括但不限於：

   - (a) 限制或終止其對 CloudXact 服務的訪問權限；
   - (b) 向相關執法機構或權利擁有者通報違規行為；
   - (c) 根據情節嚴重性永久禁止其使用本平台服務。
2. **侵權通知與申請流程**
若您認為您的版權已被侵害，請通過我們的 [幫助中心](https://support.cloudxact.com/) 提交通知，並提供以下詳細資料以協助我們進行調查與處理：

   - (a) 版權擁有者的姓名、聯絡資訊及合法身份證明文件；
   - (b) 涉嫌侵權內容的具體位置（如網站 URL 或詳細描述）；
   - (c) 涉嫌侵權行為的詳細說明，包括受保護作品的名稱或描述；
   - (d) 您聲明提供的信息準確無誤，並確認您是版權擁有者或經授權代理人。
3. **處理流程與響應時間**

   - (a) 收到完整的侵權通知後，我們將在最慢兩週的時間內進行初步審查；
   - (b) 若確認存在侵權行為，我們將採取適當措施，包括移除相關內容或限制其使用；
   - (c) 同時，我們將通知涉嫌侵權方，並提供其申訴或提交證據的機會，以確保程序公平公正。
4. **免責聲明**
   CloudXact 僅作為用戶內容的托管平台，對於用戶所發布的內容不擔保其無侵權行為。用戶應對其上傳或分享的內容負全責，並同意賠償因此對第三方造成的任何損害。

## 第四章 使用者內容管理  

1. **連結政策的管理權限**  
   CloudXact 保留以下管理權限，以確保平台的完整性與合法性：  
   - (a) 隨時移除不符合政策的連結，且不另行通知；  
   - (b) 修改現有連結政策，並視情況在網站上發布更新通知；  
   - (c) 對於屢次違規或存在潛在風險的用戶或組織，採取額外限制措施，包括暫停其使用權限。

2. **問題連結的處理程序**  
   若您發現任何連結內容涉嫌不當或違規，請透過 [幫助中心](https://support.cloudxact.com/) 並遵循以下步驟與我們聯繫：  
   - (a) 提供具體的連結 URL 或內容描述；  
   - (b) 詳細說明問題的性質，例如誤導性資訊、侵犯權利或不符政策的內容；  
   - (c) 附上相關證據（如截圖或文件），以便我們進行審查。  

   我們將根據收到的舉報進行全面評估，並採取以下可能的補救措施：  
   - 移除問題連結；  
   - 向相關用戶或組織發出警告；  
   - 必要時通知法律或執法機構協助處理。

3. **用戶的責任與承諾**  
   - (a) 用戶應確保提交至 CloudXact 平台的內容真實、合法且不侵犯任何第三方權利；  
   - (b) 用戶同意承擔因提交不當內容而引發的法律責任，包括賠償因侵權或違規行為對 CloudXact 或其他方造成的損害；  
   - (c) 用戶須遵守 CloudXact 的政策更新，並定期查看相關通知，避免因未了解更新而違規。

4. **免責聲明**  
   CloudXact 作為內容托管與管理平台，對用戶上傳的內容不負事先審查的義務，但在收到舉報後將採取合理的處理措施。我們對於因第三方內容引發的任何直接或間接損害不承擔責任，但將竭盡所能維護平台環境的安全性與合法性。

## 第五章 個人資訊的收集  

1. **基本資訊的收集**  
   為了提供最佳的服務體驗，我們在使用本服務期間可能會收集以下類型的資訊：  
   - (a) **基本身份資訊**：例如姓名、電子郵件地址、聯絡電話及其他基本聯絡資料；  
   - (b) **商業資訊**：如公司名稱、職位、業務性質等資訊，用於提供量身定制的企業服務；  
   - (c) **互動資訊**：您在使用服務過程中提供的反饋、評論、建議或其他互動內容，這些有助於我們改善服務質量。

2. **使用技術收集行為資料**  
   我們可能會使用 **Cookie** 和類似技術（如像素標籤和網絡信標）來收集您的使用資訊。這些技術有助於我們分析您的瀏覽行為、設備資訊以及您如何與我們的服務互動，從而提升服務質量、優化用戶體驗以及進行定向推廣。若您希望了解更多關於這些技術如何運作以及如何管理您的偏好，請參閱我們的 [Cookie 政策](https://www.cloudxact.com/terms/cookie-policy)。

3. **第三方資料收集**  
   除了您直接提供的資訊，我們還可能從以下第三方來源收集額外的資料，以增強我們對您的服務與需求的了解：  
   - (a) **社交媒體平台**：若您使用社交媒體帳戶與我們互動，我們可能會獲取您的公開資料，如公開的社交媒體帳號資料；  
   - (b) **公開數據庫**：我們可能會從公開可訪問的資料庫中獲取有關您或您的組織的公開資訊，這些資訊有助於我們提供更準確的服務建議或推廣內容。

4. **資訊收集的合法性與透明度**  
   我們承諾僅在符合法律要求且基於您的同意或合法利益的情況下收集您的個人資料。我們會確保所有收集的資訊清晰、透明，並且僅用於特定的目的。

## 第六章 個人資訊的使用  

1. **個人資訊的使用目的**  
   我們將根據以下目的使用您的個人資訊，旨在提供高品質的服務並提升用戶體驗：  
   - (a) **提供服務與處理訂單**：根據您的需求提供服務，並確保順暢的訂單處理流程，為您提供個性化的服務體驗；  
   - (b) **回應查詢與解決技術問題**：對您的查詢進行快速回應，並提供技術支持，解決您在使用服務過程中可能遇到的問題；  
   - (c) **發送產品與服務相關資訊**：定期或根據需求向您發送有關我們的產品、服務、促銷活動及其他行銷資訊；  
   - (d) **市場研究與服務改進**：進行市場調查、用戶反饋收集，並基於分析結果優化服務流程與體驗；  
   - (e) **遵守法規要求與確保合規性**：處理您的資料以符合法律、法規或監管機構的要求，並確保我們的服務保持合規。

2. **不出售您的個人資訊**  
   除非法律要求或您明確同意，我們不會出售、出租或與任何第三方共享您的個人資訊，也不會將您的個人資訊用於未經授權的目的。我們的服務使用僅限於以上明示的目的，並將竭力保護您的隱私權。

3. **與第三方的資料共享**  
   在以下情況下，我們可能會與子公司、合作夥伴、服務供應商共享您的必要個人資訊：  
   - (a) **服務提供**：我們可能與業務合作夥伴協作來提供您所需的產品或服務（如支付處理商、物流公司等）；  
   - (b) **法律或監管要求**：在需要時，為符合法律、法規、司法命令或政府要求，我們可能必須披露您的資料；  
   - (c) **數據處理協議**：我們將與第三方服務提供商簽訂數據處理協議，以確保您的個人資訊受到適當的安全保障與保密措施。

4. **自定義資料共享選項**  
   根據您的需求，我們提供可選的自定義選項，讓您可以決定是否允許共享您的個人資訊，以及該資料共享的具體範圍和用途。這些選項將根據您的個人偏好進行設置，並且您可以隨時修改。

## 第七章 資料安全保障

1. **資訊安全保護措施**  
   我們致力於保護您的資料免受未經授權的訪問、披露、竄改或銷毀。為此，我們採用了多種技術和組織措施來保障資訊安全，包括但不限於：  
   - (a) **高強度加密技術**：所有敏感資料在傳輸和存儲過程中均進行加密處理，使用先進的加密技術來保護您的資料安全；  
   - (b) **訪問控制與權限管理**：實施嚴格的訪問控制，僅授權的員工可以訪問必要的資料，並定期檢查與更新權限設置；  
   - (c) **定期安全審核與漏洞檢測**：定期進行安全審核，並透過漏洞掃描來識別並修補系統中的安全漏洞；  
   - (d) **模擬攻擊測試**：定期進行模擬攻擊測試（如紅隊演練）以評估我們系統的防禦能力，確保能夠抵抗各種潛在威脅。

2. **資料外洩應對與通知**  
   在極少數情況下，若發生資料外洩事件，我們將採取以下行動：  
   - (a) **迅速補救措施**：一旦發現資料外洩，我們將立即啟動應急響應程序，採取適當的補救措施，將損害降到最低；  
   - (b) **依法律通知受影響方**：我們將根據法律要求，及時通知受影響的用戶和監管機構，並提供後續支持，協助用戶應對潛在風險。

3. **用戶安全責任**  
   我們為確保資料安全，提供了多重保障措施，但用戶自身亦有責任保護其帳戶安全，包括：  
   - (a) **妥善管理帳號密碼**：用戶應定期更新密碼，並使用強密碼（包含字母、數字與特殊符號）來減少帳戶被破解的風險；  
   - (b) **避免在公共網絡上提交敏感資料**：在公共或不安全的網絡環境中，請避免輸入或提交敏感資訊，如密碼、支付信息等，避免潛在的資料泄露風險。

## 第八章 資料跨境傳輸

1. **資料跨境傳輸原則**  
   在某些情況下，為了提供服務或履行法律義務，我們可能需要將您的資料傳輸至其他法域。無論資料傳輸至何地，我們都將確保遵循嚴格的資料保護措施，並符合法律規範及最佳實踐標準。例如，對於歐盟內部的資料傳輸，我們將遵守歐盟的一般資料保護條例（GDPR），以確保您的資料得到同等的保護。

2. **資料共享與安全保障**  
   資料共享僅在法律允許的情況下進行，或經您明確同意。為保障資料在跨境傳輸過程中的安全，我們將採取包括加密等技術性安全保障措施，防止資料在傳輸過程中遭受未經授權的訪問或損害。

3. **跨境資料傳輸合規性考量**  
   在進行跨境資料傳輸時，我們將充分考慮各地隱私法律及其要求的差異，並依據目的地國家的隱私保護標準，進行必要的調整與合規保障。無論您的資料處於何處，我們始終會確保其得到最高水準的保護，並根據不同法域的要求，採取適當的法律與技術措施，以維護您的隱私權。

## 第九章 未成年人的隱私

1. **未成年用戶資料保護原則**  
   我們致力於保護未成年人的隱私，並不主動收集未滿 18 歲者的個人資訊。我們已採取技術及管理措施，防止未成年人使用我們的服務，並確保未成年用戶的資料不會被收集或存儲。

2. **未成年用戶資料的審查與刪除**  
   若我們發現收集到未滿 18 歲用戶的資料，將立即採取行動進行審核，並依據情況迅速刪除相關數據。我們對此類數據的處理將符合適用的法律規範，並確保未成年用戶的資料安全。

3. **處理未成年用戶資料的疑慮**  
   若您認為您的子女或未成年用戶在未經許可的情況下向我們提供了個人資訊，請立即聯繫我們。我們將盡快採取適當的行動，包括刪除相關資料並提供必要的支持。

1. **政策變更通知**  
   我們會根據業務需求或法律法規的變動，定期更新本隱私政策。每次更新後，我們將在網站上發布公告，並通過適當的渠道（如電子郵件及應用內通知）通知您，以確保您能及時了解並理解政策的變更內容。

2. **重大變更處理**  
   若本政策的變更涉及重大內容或對您的權利及責任產生實質性影響，我們將提供專門的簡報、互動頁面或其他形式的溝通方式，幫助您快速理解政策調整及其潛在影響。

3. **繼續使用服務即視為同意**  
   在政策更新後，若您繼續使用我們的服務，即表示您接受更新後的政策條款。如果您不同意更新的內容，您有權隨時停止使用服務，並根據需要聯繫我們以了解更多信息。
